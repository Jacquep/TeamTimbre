# Timbre

### Synopsis
***

A JavaScript local live music event discovery application that uses the Eventful api to search for local live music.

***

#### Firebase is used to store emails and passwords so that you can save your selected events to a list 

![login-page](https://user-images.githubusercontent.com/17747867/29744367-81e664da-8a58-11e7-9889-efc017ec93ad.png)

![listpage](https://user-images.githubusercontent.com/17747867/29744364-81e35286-8a58-11e7-9f0e-047f6b3e88da.png)

#### Once you are logged in, you are taken to the event listing page. At the top of the page you can zero in on the neighborhood where you would like to see your event. 

![results-page](https://user-images.githubusercontent.com/17747867/29744368-81e6bda4-8a58-11e7-8c36-bd53c1ea2287.png)

#### When you select an event more information about the artist and event is displayed to your right.  

![selected-view](https://user-images.githubusercontent.com/17747867/29744363-81e2ee86-8a58-11e7-87a6-482328551d19.png)

#### When you select the heart, the event is added to your list where all of your selected events are saved. 

![mylist](https://user-images.githubusercontent.com/17747867/29744366-81e42350-8a58-11e7-9c10-01d13ace5ee9.png)
